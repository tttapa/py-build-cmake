# Installation

## PyPI

The py-build-cmake package is available on
[PyPI](https://pypi.org/project/py-build-cmake/):

```sh
pip install py-build-cmake
```

## From source

py-build-cmake can build itself:

```sh
git clone https://github.com/tttapa/py-build-cmake.git
cd py-build-cmake
pip install .
```
